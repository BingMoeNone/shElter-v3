# 项目需求文档

## 项目名称
**数字避难所 | shElter**

---

## 项目概述

这是一个数字人物、数字生态的收录 wiki 平台，类似于 SCP 基金会的创作模式。用户可以创作完整的角色、世界观，与同好联动、交友。

**核心定位：** 提供一个开放、协作的数字创作平台，让创作者能够分享、展示和协作创作数字内容。

---

## 功能性需求

### 1. 内容创作与管理

#### 1.1 文档创作
- **创建文档**：用户可以创建角色、世界观等类型的文档
- **编辑文档**：所有用户都可以编辑文档，支持多人协作
- **版本控制**：记录文档的编辑历史，支持版本对比和回滚
- **发布管理**：支持草稿和已发布状态
- **删除管理**：作者和管理员可以删除文档

#### 1.2 媒体管理
- **图片上传**：支持上传图片作为文档附件或内容插入
- **图片管理**：管理用户上传的图片，支持查看、删除
- **图片嵌入**：在文档中嵌入上传的图片
- **图片预览**：上传前支持图片预览
- **图片大小限制**：限制单张图片大小，支持批量上传

#### 1.3 分类与标签
- **分类管理**：支持多级分类结构
- **标签系统**：为文档添加标签，支持热门标签
- **分类筛选**：按分类筛选文档
- **标签搜索**：通过标签搜索相关文档

### 2. 用户系统

#### 2.1 用户认证
- **注册功能**：支持用户名和邮箱注册
- **登录功能**：用户名/邮箱 + 密码登录，未登录可以浏览文档但不可编辑。
- **密码管理**：修改密码、重置密码
- **JWT 认证**：使用 JWT 进行身份验证

#### 2.2 用户信息
- **个人资料**：管理用户头像、昵称、简介等信息
- **创作统计**：展示用户的创作数量、获得的点赞数等
- **关注系统**：关注其他用户，查看关注列表和粉丝列表

#### 2.3 权限管理
- **编辑权限**：所有用户可编辑文档
- **删除权限**：作者和管理员可删除文档
- **管理权限**：管理员可管理用户和系统设置

### 3. 社交互动

#### 3.1 评论系统
- **添加评论**：对文档和其他评论进行评论
- **嵌套评论**：支持评论的回复功能
- **评论管理**：作者和管理员可删除评论
- **评论点赞**：对评论进行点赞

#### 3.2 内容互动
- **点赞功能**：对文档进行点赞
- **收藏功能**：收藏感兴趣的文档
- **分享功能**：分享文档到外部平台

#### 3.3 通知系统
- **评论通知**：收到评论时的通知
- **点赞通知**：收到点赞时的通知
- **关注通知**：被关注时的通知
- **系统通知**：系统相关的通知

### 4. 内容发现与搜索

#### 4.1 推送系统
- **今日推荐**：展示当日最新创作的文档
- **热门推荐**：展示近期热门文档
- **个性化推荐**：基于用户兴趣的推荐
- **新鲜内容**：展示近期更新的内容

#### 4.2 搜索功能
- **全文搜索**：搜索文档标题和内容
- **高级搜索**：按分类、标签、作者等条件搜索
- **搜索历史**：记录用户的搜索历史
- **热门搜索**：展示热门搜索关键词

#### 4.3 文档管理
- **文档列表**：按分类展示所有文档
- **文档排序**：支持按时间、热度、相关度排序
- **文档筛选**：按类型、状态等筛选文档
- **文档预览**：在列表中展示文档摘要

### 5. 技术支持

#### 5.1 API 接口
- **RESTful API**：遵循 RESTful 设计规范
- **API 文档**：提供 Swagger UI 和 ReDoc 文档
- **统一响应**：统一的 API 响应格式
- **错误处理**：完善的错误处理机制

#### 5.2 数据安全
- **密码加密**：使用 bcrypt 加密存储密码
- **Token 认证**：使用 JWT 进行身份验证
- **请求限制**：Rate Limiting 防止滥用
- **数据备份**：定期备份数据库

## 后端功能需求

### 1. 内容管理

#### 1.1 文档管理
- **CRUD 操作**：创建、读取、更新、删除文档
- **版本控制**：记录每次编辑，支持版本对比和回滚
- **状态管理**：草稿、已发布、已删除状态
- **权限控制**：基于用户角色的权限控制

#### 1.2 媒体管理
- **图片上传**：支持图片上传和存储
- **图片处理**：图片压缩、格式转换
- **图片管理**：查看、删除上传的图片
- **图片访问**：提供图片访问接口

#### 1.3 分类与标签
- **分类管理**：创建、编辑、删除分类
- **标签管理**：创建、编辑、删除标签
- **关联管理**：文档与分类、标签的关联

### 2. 用户管理

#### 2.1 认证系统
- **注册**：用户注册，邮箱验证
- **登录**：用户登录，JWT 生成
- **Token 管理**：Token 刷新、失效
- **密码管理**：密码修改、重置

#### 2.2 用户信息
- **资料管理**：用户资料的更新和查询
- **统计信息**：用户创作、互动统计
- **关注系统**：关注关系的管理

#### 2.3 权限控制
- **角色管理**：普通用户、管理员、超级管理员
- **权限分配**：基于角色的权限分配
- **访问控制**：API 访问权限控制

### 3. 社交系统

#### 3.1 评论管理
- **评论 CRUD**：创建、读取、更新、删除评论
- **嵌套评论**：支持评论的回复
- **评论统计**：评论数量、点赞数

#### 3.2 互动管理
- **点赞系统**：文档和评论的点赞
- **收藏系统**：文档的收藏
- **分享系统**：文档的分享记录

#### 3.3 通知系统
- **通知生成**：自动生成各类通知
- **通知管理**：通知的查询、标记已读
- **通知推送**：实时通知推送

### 4. 搜索与推荐

#### 4.1 搜索系统
- **全文搜索**：文档内容的全文搜索
- **索引优化**：建立搜索索引，支持中文分词
- **高级搜索**：多条件组合搜索
- **搜索统计**：搜索关键词统计

#### 4.2 推荐系统
- **热门推荐**：基于热度的推荐
- **新鲜推荐**：基于时间的推荐

### 5. 系统管理

#### 5.1 数据统计
- **内容统计**：文档数量、分类统计
- **用户统计**：用户数量、活跃度
- **互动统计**：评论、点赞、收藏统计

#### 5.2 安全管理
- **请求限制**：Rate Limiting 配置
- **日志记录**：操作日志、访问日志
- **数据备份**：定期数据库备份

#### 5.3 API 管理
- **API 文档**：自动生成 API 文档
- **API 版本**：API 版本控制
- **API 监控**：API 访问监控
