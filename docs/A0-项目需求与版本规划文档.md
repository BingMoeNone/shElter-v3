# A0-项目需求与版本规划文档

## 项目名称
**数字避难所 | shElter**

---

## 项目概述

这是一个用于收录架空世界观的 Wiki 平台，用户可以创作属于自己的架空世界观及角色，与同好交友讨论、与别的作品联动。

**核心定位：** 提供一个开放、协作的架空世界创作收录平台，让创作者能够分享、展示和协作创作相关内容。

**对标类比：** 本平台对标 Fandom，收录的架空世界观类比 Fandom 平台内的所有 Wiki，用户可以自行创建、组建社群管理。

---

## 核心功能设计

### 1. 首页设计

#### 1.1 核心功能
- **推送流**：展示最新更新的文档和活动
- **热门推荐**：展示近期热门的世界观和文档
- **个性化推荐**：基于用户兴趣推荐相关内容
- **快捷入口**：快速访问常用功能（创建文档、探索世界、个人中心）


---

### 2. 用户页面

#### 2.1 核心功能
- **个人资料展示**：头像、昵称、简介、等级
- **社交关系**：关注列表、粉丝列表
- **创作展示**：创作的作品文档列表
- **角色设定**：自己给自己设计的角色设定展示
- **履历时间线**：按时间顺序展示浏览、关注过的世界观和文档
- **贡献统计**：创作数量、获得点赞、评论数等


---

### 3. 探索页面

#### 3.1 核心功能
- **世界观收录**：以地图的形式展示所有架空世界观
- **分类浏览**：按类型、标签筛选世界观
- **搜索功能**：全文搜索世界观和文档
- **编辑权限**：登录用户可编辑文档
- **用户组管理**：每个世界观可设置不同用户组，不同用户组权限不同
- **社群管理**：世界观创建者可组建社群、管理成员


---

### 4. 世界观详情页

#### 4.1 核心功能
- **世界观信息**：名称、简介、封面、创建者
- **文档管理**：世界观下的所有文档列表
- **成员管理**：世界观成员列表和权限
- **用户组设置**：不同用户组的权限配置
- **活动记录**：世界观的更新历史



---

### 5. 文档编辑页

#### 5.1 核心功能
- **富文本编辑**：支持 Markdown 或富文本编辑器
- **图片上传**：支持插入图片
- **版本历史**：保存文档修订历史
- **协作编辑**：多人协作编辑文档
- **评论讨论**：文档内评论和讨论

## 技术实现建议

### 数据库设计要点

#### 核心表结构
```
users - 用户表
├── id, username, email, password_hash
├── avatar_url, bio, display_name
├── created_at, updated_at
└── role (user, moderator, admin)

worlds - 世界观表
├── id, name, slug, description
├── cover_image, creator_id
├── created_at, updated_at
└── status (active, archived)

documents - 文档表
├── id, title, slug, content
├── world_id, author_id
├── status (draft, published, archived)
├── view_count, like_count
└── created_at, updated_at

world_members - 世界观成员表
├── id, world_id, user_id
├── role (owner, admin, editor, member)
└── joined_at

comments - 评论表
├── id, document_id, author_id
├── content, parent_id (嵌套评论)
└── created_at

follows - 关注表
├── id, follower_id, followed_id
└── created_at
```

### API 设计建议

#### 核心接口
```
# 认证
POST   /api/auth/register
POST   /api/auth/login
POST   /api/auth/logout
GET    /api/auth/me

# 用户
GET    /api/users/:username
PUT    /api/users/me
GET    /api/users/:username/followers
GET    /api/users/:username/following
POST   /api/users/:username/follow
DELETE /api/users/:username/follow

# 世界观
GET    /api/worlds
POST   /api/worlds
GET    /api/worlds/:id
PUT    /api/worlds/:id
DELETE /api/worlds/:id
GET    /api/worlds/:id/members
POST   /api/worlds/:id/members

# 文档
GET    /api/worlds/:id/documents
POST   /api/worlds/:id/documents
GET    /api/documents/:id
PUT    /api/documents/:id
DELETE /api/documents/:id
GET    /api/documents/:id/history

# 评论
GET    /api/documents/:id/comments
POST   /api/documents/:id/comments
DELETE /api/comments/:id

# 搜索
GET    /api/search?q=keyword&type=worlds|documents
```
