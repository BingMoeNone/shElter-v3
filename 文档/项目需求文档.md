# 项目需求文档

## 项目名称
**数字避难所 | shElter**

---

## 项目概述

这是一个数字人物、数字生态的收录 wiki 平台，类似于 SCP 基金会的创作模式。用户可以创作完整的角色、世界观，与同好联动、交友。

**核心定位：** 提供一个开放、协作的数字创作平台，让创作者能够分享、展示和协作创作数字内容。

---

## 总体需求

### 1. 核心功能
- **内容创作**：用户可以创建文档、编辑文档、以文档的形式发布原创角色、原创世界观等数字内容
- **协作编辑**：所有文档用户都可编辑，支持多人协作
- **社交互动**：用户可以关注、评论、与其他创作者联动
- **内容发现**：通过推送和搜索发现优质内容
- **个人展示**：展示用户的个人资料和所有创作内容
- **作品管理**：所有作品都将获得一个唯一坐标，以地图形式展出所有作品，所有相近的文档坐标也相近。

### 2. 项目架构
整个项目可以分为三个核心部分：

#### 个人部分
- 储存和展示用户的 ID 和所有创作的内容
- 包括角色、世界观、评论等
- 个人主页展示用户的创作历程和作品集

#### 推送页面
- 推送他人的优质文档
- 展示本日或近期创作的最新文档
- 基于用户兴趣的个性化推荐

#### 文档管理页面
- 按序收录所有文档，分区明确：角色、武器、道具、概念、世界观、建筑、生态环境。
- 所有文档用户都可编辑
- 用户也可以新增文档
- 检索功能要完善，支持多维度搜索
- 文档支持标签分类和插入别的文档的链接

### 3. 设计原则
- 页面简洁，结构清晰
- 设计独特，体现数字生态的特色
- 注重用户体验，操作流畅
- 响应式设计，支持多端访问

## 前端需求

### 1. 核心页面

#### 1.1 首页 / 推送页面
- **功能需求**：
  - 展示今日最新创作的文档列表
  - 展示近期热门文档
  - 展示推荐文档（基于用户兴趣）
  - 支持文档预览（摘要、标签、作者信息）
  - 支持快速跳转到文档详情页
  - 支持筛选（按时间、热度、分类等）

#### 1.2 个人主页
- **功能需求**：
  - 展示用户基本信息（头像、昵称、简介等）
  - 展示用户创作的所有内容（角色、世界观等）
  - 展示用户的评论历史
  - 展示用户的关注列表和粉丝列表
  - 支持编辑个人资料
  - 支持查看用户的创作统计

#### 1.3 文档详情页
- **功能需求**：
  - 展示文档完整内容
  - 展示文档元信息（作者、创建时间、最后编辑时间、版本历史、标签）
  - 支持文档编辑（所有用户）
  - 支持查看文档编辑历史
  - 展示文档评论列表
  - 支持添加评论
  - 支持收藏文档
  - 支持分享文档

#### 1.4 文档管理页
- **功能需求**：
  - 按分类展示所有文档
  - 支持新增文档
  - 支持搜索文档（按标题、内容、标签、作者等）
  - 支持高级筛选（按时间、热度、分类等）
  - 支持文档排序
  - 展示文档分区（如：角色区、世界观区等）

#### 1.5 文档编辑页
- **功能需求**：
  - 富文本编辑器
  - 支持插入图片、链接、表格等
  - 支持添加标签
  - 支持选择分类
  - 支持保存草稿
  - 支持预览
  - 支持版本对比

#### 1.6 搜索页
- **功能需求**：
  - 搜索框（支持全文搜索）
  - 搜索结果展示
  - 高级搜索选项（按类型、时间、作者等）
  - 搜索历史记录
  - 热门搜索推荐

#### 1.7 用户登录/注册页
- **功能需求**：
  - 用户名/邮箱登录
  - 密码登录
  - 新用户注册
  - 忘记密码
  - 第三方登录（可选）

### 2. 通用功能

#### 2.1 导航系统
- 顶部导航栏（首页、文档、个人、搜索等）
- 面包屑导航
- 侧边栏导航（文档分类等）

#### 2.2 用户交互
- 点赞/收藏功能
- 评论功能
- 关注/取关功能
- 分享功能

#### 2.3 通知系统
- 系统通知
- 评论通知
- 关注通知
- 点赞通知

#### 2.4 搜索功能
- 全局搜索
- 分类搜索
- 高级搜索

## 本需求文档仅给出功能与接口要求，页面布局、交互流程及视觉结构可由实现者自由设计。

## 后端需求

### 1. 用户系统

#### 1.1 用户认证
- **注册功能**：
  - 用户名注册（唯一性校验）
  - 邮箱注册（唯一性校验）
  - 密码加密存储（bcrypt）
  - 邮箱验证（可选）

- **登录功能**：
  - 用户名/邮箱 + 密码登录
  - JWT Token 认证
  - Token 刷新机制
  - 记住登录状态

- **密码管理**：
  - 修改密码
  - 重置密码（通过邮箱）
  - 密码强度验证

#### 1.2 用户信息
- **用户资料**：
  - 用户 ID（唯一标识）
  - 用户名
  - 邮箱
  - 头像
  - 个人简介
  - 注册时间
  - 最后登录时间

- **用户统计**：
  - 创作数量（角色、世界观等）
  - 评论数量
  - 获得点赞数
  - 粉丝数量
  - 关注数量

#### 1.3 权限管理
- **用户角色**：
  - 普通用户
  - 管理员
  - 超级管理员

- **权限控制**：
  - 文档编辑权限（所有用户）
  - 文档删除权限（作者、管理员）
  - 用户管理权限（管理员）
  - 系统配置权限（超级管理员）

### 2. 内容系统

#### 2.1 文档管理
- **文档类型**：
  - 角色文档
  - 世界观文档
  - 其他自定义类型

- **文档属性**：
  - 文档 ID（唯一标识）
  - 标题
  - 内容（富文本）
  - 摘要
  - 作者 ID
  - 分类 ID
  - 标签列表
  - 创建时间
  - 最后编辑时间
  - 最后编辑者 ID
  - 版本号
  - 状态（草稿、已发布、已删除）

- **文档操作**：
  - 创建文档
  - 编辑文档（所有用户）
  - 删除文档（作者、管理员）
  - 发布/取消发布
  - 恢复文档

#### 2.2 版本控制
- **版本记录**：
  - 记录每次编辑
  - 保存版本快照
  - 记录编辑者信息
  - 记录编辑时间

- **版本操作**：
  - 查看版本历史
  - 版本对比
  - 恢复到指定版本

#### 2.3 分类系统
- **分类属性**：
  - 分类 ID（唯一标识）
  - 分类名称
  - 父分类 ID（支持层级）
  - 描述
  - 排序权重

- **分类操作**：
  - 创建分类
  - 编辑分类
  - 删除分类
  - 查询分类树

#### 2.4 标签系统
- **标签属性**：
  - 标签 ID（唯一标识）
  - 标签名称
  - 使用次数
  - 创建时间

- **标签操作**：
  - 创建标签
  - 编辑标签
  - 删除标签
  - 热门标签查询

### 3. 社交系统

#### 3.1 关注系统
- **关注关系**：
  - 关注者 ID
  - 被关注者 ID
  - 关注时间

- **关注操作**：
  - 关注用户
  - 取消关注
  - 查询关注列表
  - 查询粉丝列表

#### 3.2 评论系统
- **评论属性**：
  - 评论 ID（唯一标识）
  - 文档 ID
  - 用户 ID
  - 评论内容
  - 父评论 ID（支持嵌套）
  - 创建时间
  - 最后编辑时间
  - 点赞数

- **评论操作**：
  - 添加评论
  - 编辑评论（作者）
  - 删除评论（作者、管理员）
  - 查询评论列表
  - 点赞评论

#### 3.3 点赞系统
- **点赞记录**：
  - 点赞者 ID
  - 目标类型（文档、评论）
  - 目标 ID
  - 点赞时间

- **点赞操作**：
  - 点赞
  - 取消点赞
  - 查询点赞列表

#### 3.4 收藏系统
- **收藏记录**：
  - 用户 ID
  - 文档 ID
  - 收藏时间

- **收藏操作**：
  - 收藏文档

  - 查询收藏列表

### 4. 搜索系统

#### 4.1 全文搜索
- **搜索范围**：
  - 文档标题
  - 文档内容
  - 文档标签
  - 用户名

- **搜索功能**：
  - 关键词搜索
  - 模糊搜索
  - 高级搜索（多条件组合）
  - 搜索结果排序（相关度、时间、热度）

#### 4.2 搜索优化
- **索引优化**：
  - 建立全文索引

- **搜索统计**：
